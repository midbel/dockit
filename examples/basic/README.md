These examples show basic usage of dockit.

The v0.dk script demonstrates usage of:

1. loading file (`import`)
2. referencing cells
3. adding columns
4. cell assignment
5. usage of builtin functions (`lower`, `concatenate`)
6. printing the result on stdout (`print`)

The v1.dk script is essentially a refactored version of v0.dk. It does the exact same thing but in fewer lines of code.
